# Experiment 4: Implement State Management using Redux / Context API

## Overview

Unit-4 experiments demonstrating different approaches to global state management in React applications.

## Experiments

### 4.1 - Global State Management Using React Context API

Implementing a Theme Switcher application to share global state across components without prop drilling using `createContext` and `useContext`.

- **Aim**: To implement global state management in a Single Page Application using the React Context API.
- **Key Concepts**: `createContext`, `Provider`, `useContext`.

### 4.2 - State Management Using Redux (Counter Application)

Implementing a centralized state container for a Counter application using Redux Toolkit.

- **Aim**: To implement centralized state management using Redux in a React application.
- **Key Concepts**: `configureStore`, `createSlice`, `useSelector`, `useDispatch`.
